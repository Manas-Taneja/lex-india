# Wiki Schema

## Topic Page Rules
1. Every topic page MUST be self-contained. No "see IPC §420" without inline explanation.
2. Current law first. Superseded law goes in Transition Note section only.
3. Plain English definition before legal text.
4. Transition notes MUST cite source: official MHA comparative tables.
5. Related topics listed as links, max 5.

## Act Summary Rules
1. One paragraph summary only.
2. Key sections listed with section number + one-line description.
3. Status: in-force / superseded / partially-superseded.

## INDEX.md Rules
1. Every topic page MUST have an entry.
2. Entry format: `- [topic](topics/X.md) — one-line description (primary act §range)`
3. Grouped by category. Categories: Criminal, Civil, Commercial, Constitutional, Labour, Tax, Family, Property, Procedure, Transitions.
