# Contributing to lex-india

## What lives where
- `acts/` — raw scraped text. Do not edit manually. File issues for scraper bugs.
- `wiki/topics/` — topic pages. PRs welcome. Must follow schema.md rules.
- `wiki/transitions/` — transition maps. Must cite official MHA sources.
- `data/` — machine-readable source data. Do not edit; generated by scripts.

## Adding a topic page
1. Check INDEX.md — does the topic exist?
2. If not: copy wiki/topics/_template.md, fill it out, add entry to INDEX.md.
3. PR title: `wiki: add topic/theft`

## Fixing a wiki page
1. Cite your source (act + section, or MHA table).
2. PR title: `wiki: fix topic/murder - update BNS section refs`
